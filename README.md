# Exchange App

A fully functional currency converter web application built using **HTML, CSS, JavaScript, Tailwind CSS, and Vite**. The application allows users to convert amounts between different currencies using real-time exchange rate data from an external API.

**Live Demo:** https://exchange-app.adeelalam10100.workers.dev/

## Features

* Real-time currency conversion using live exchange rates
* Support for multiple international currencies
* Currency swap functionality
* Responsive and modern user interface
* Input validation and user-friendly interactions
* Country + Currency name displayed for selected currencies

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Tailwind CSS
* Vite
* Exchange Rate API

## Development & Deployment

The project was developed using the Vite development server for a fast and efficient development experience. After completion, it was optimized for production using Vite's build process and deployed on Cloudflare Workers. The complete source code is available on GitHub for version control and collaboration.

This project demonstrates practical frontend development skills, including API integration, asynchronous JavaScript, DOM manipulation, responsive UI design, modern frontend tooling with Vite, and deployment of a production-ready web application.
